## Name

S.A.D.E. (Style A Design Experience)

## Description

S.A.D.E. is a CSS Design System: UI for cards, buttons, forms, navigation, typography and other UI components.


## MVP Features:

• Header

• Navigation

• Flexbox for Layout

• Dark Mode

• Cards

• Buttons

• Currently working on 3 Styles:
- Stronger
- Cherish
- Taboo
